# DESIGN — Glossary

> Living document. Project-internal terminology, abbreviations, and
> in-house words. When a term is used in multiple docs with project-
> specific meaning, define it here.

---

## 1. Conventions

- Entries alphabetized.
- Each entry: term, definition, cross-doc references.
- Mark **(canonical PF2e)** when a term has the upstream PF2e meaning.
- Mark **(reworked)** when we've changed an upstream term's meaning.
- Mark **(project)** when the term is invented by us.

## 2. Glossary (seed; expand as we go)

- **AP** **(canonical)** — Adventure Path.
- **Backlog** **(project)** — actionable-task list, lives in `Backlog/`.
- **Body state** **(project)** — see *Body and Identity*.
- **Canonical body** **(project)** — see *NSFW* §5.1; the fixed-by-us
  baseline body.
- **Canonical tier** **(project)** — see *Content Pipeline*.
- **Content extract** **(project)** — a piece of source material being
  imported.
- **DESIGN folder** **(project)** — `DESIGN/` at project root.
- **Diegetic** **(industry)** — UI/audio that exists within the fiction.
- **Edicts / Anathema** **(canonical)** — what a deity requires /
  forbids.
- **Essence-tier reagents** **(project)** — see *Magic* §1.1.

- **Gherman** **(project)** — the protagonist (see *Vision* §4.1).

  adopting Remaster RAW where it differs from prior editions

- **Hot cache** **(project)** — in-memory game-side data layer (see
  *Tech and Engine* §3).
- **Iconic Heroes** **(canonical)** — Paizo's pre-built sample
  characters.
- **L0 / L1 / L2 / L3 / L4** **(project)** — simulation tick layers
  (see *World Simulation* §3).
- **MAP** **(canonical)** — Multiple Attack Penalty.
- **NSFW asset filter** **(project)** — see *NSFW* §6 + N-Q13.
- **Open Question** **(project)** — see *Workflow* §5.
- **PF2e** — Pathfinder Second Edition.
- **PF2e Remaster** — the 2023+ rules update under ORC.
- **Plugin** **(project)** — see *Modding and Extensibility*.
- **Project rework** **(project)** — see *Lore Reworks*.
- **RAW** — rules as written.
- **Reagent** **(reworked)** — see *Magic* §1.1; required for casting

- **Source tier** **(project)** — see *Content Pipeline*.
